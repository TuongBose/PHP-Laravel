Nếu đã tồn tại controller này rồi thì sẽ hiện thông báo controller này đã tồn tại
ERROR  Controller already exists.

Lệnh tạo controller cưỡng bức, bắt buộc, không kiểm tra đã tồn tại hay chưa, controller đã tồn tại sẽ bị ghi đè
php artisan make:controller ProductController --force

Lệnh php artisan storage:link trong Laravel dùng để tạo một symbolic link (liên kết tượng trưng) từ thư mục public/storage trỏ tới thư mục storage/app/public.
Cụ thể:
Trong Laravel, khi bạn lưu file (ảnh, tài liệu...) bằng disk public (định nghĩa trong config/filesystems.php), thì file sẽ được lưu ở:
storage/app/public

Nhưng thư mục storage/app/public không truy cập trực tiếp từ web được.
Lệnh php artisan storage:link sẽ tạo một symbolic link trong thư mục public/ như sau:
public/storage  --->  storage/app/public

Nhờ vậy, bạn có thể truy cập file đã upload thông qua URL, ví dụ:
http://localhost/storage/tenfile.jpg

php artisan storage:link
